# Docker-net-profiler
About Docker Network Profiler – Python-утилита для анализа контейнеров и сетевых соединений на локальном Docker-хосте. Она собирает данные через Docker API и /proc/net, отслеживает трафик, активные связи и остановленные контейнеры. Результаты сохраняются в HTML, Markdown и JSON с наглядной схемой взаимодействия.
